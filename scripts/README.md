## 🔧 可用脚本

### 版本检查脚本

**文件**：`version-check.sh`

**用途**：检查技术组件的版本发布状态

**功能**：

- 检查 Kubernetes 版本发布状态
- 检查 K3s 版本发布状态
- 检查 WasmEdge 版本发布状态
- 检查 Gatekeeper 版本发布状态

**使用方法**：

```bash
# 赋予执行权限
chmod +x scripts/version-check.sh

# 运行脚本
./scripts/version-check.sh
```

**输出示例**：

```text
[INFO] 开始技术版本检查...

[INFO] 检查 Kubernetes 1.30 发布状态...
✅ Kubernetes 1.30 已发布
"tag_name":"v1.30.0"

[INFO] 检查 K3s 1.30 发布状态...
✅ K3s 1.30 已发布
"tag_name":"v1.30.4+k3s1"

[INFO] 检查 WasmEdge 0.14 发布状态...
✅ WasmEdge 0.14 已发布
"tag_name":"0.14.0"

[INFO] 检查 Gatekeeper 3.15 发布状态...
⚠️  Gatekeeper 3.15 未找到或未发布

[INFO] 版本检查完成
```

## 📝 脚本开发规范

### 脚本结构

1. **Shebang**：`#!/bin/bash`
2. **错误处理**：`set -e`
3. **颜色定义**：使用 ANSI 颜色代码
4. **日志函数**：`log_info`、`log_warn`、`log_error`
5. **主函数**：`main()`

### 代码风格

- 使用 4 空格缩进
- 函数名使用小写字母和下划线
- 变量名使用小写字母和下划线
- 添加注释说明关键逻辑

### 错误处理

- 使用 `set -e` 在错误时退出
- 使用 `return 0` 表示成功
- 使用 `return 1` 表示失败
