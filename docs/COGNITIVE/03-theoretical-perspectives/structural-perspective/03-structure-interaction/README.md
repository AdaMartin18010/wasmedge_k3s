## 12.3.2 复合结构分析

### 核心思想

正如数学中可以有"拓扑群"（拓扑+群），计算机科学中结构也是**复合**的：

| 复合结构           | 组成                | 例子                                         |
| ------------------ | ------------------- | -------------------------------------------- |
| **函数式并发模型** | 计算结构 + 控制结构 | 纯函数（代数） + Actor 模型（消息序）        |
| **类型化 λ-演算**  | 计算结构 + 信息结构 | λ-演算（代数） + 类型系统（信息拓扑）        |
| **拓扑持续计算**   | 信息结构 + 控制结构 | 在动态数据流上维护拓扑特征（如持续同调更新） |
| **反应式编程**     | 控制结构 + 信息结构 | 时间变化的数据流（控制） + 信号近似（信息）  |

> 结构复合 ⇒ 系统复杂性与表达能力提升

**详细文档**：[01-composite-structures.md](01-composite-structures.md)

## 12.3.3 结构间关系

### 深层联系

三类结构之间存在深层联系：

1. **Curry-Howard 同构**：计算结构（程序） ↔ 信息结构（证明）
2. **分布式一致性**：控制结构（偏序） + 信息结构（容错逼近） ⇒ CAP 定理
3. **神经网络可解释性**：信息结构（嵌入空间拓扑） + 计算结构（函数复合） ⇒ 决策
   边界分析

**详细文档**：[02-structure-relationships.md](02-structure-relationships.md)
