## 12.3.2.1 Curry-Howard 同构

### 定义

**计算结构（程序） ↔ 信息结构（证明）**:

这是计算机科学中最深刻的联系之一。

### 对应关系

| 计算结构（程序） | 信息结构（证明） |
| ---------------- | ---------------- |
| 类型             | 命题             |
| 程序             | 证明             |
| 函数组合         | 逻辑推理         |
| 函数类型         | 蕴含             |
| 积类型           | 合取             |
| 和类型           | 析取             |

### 示例

```haskell
-- 类型签名（命题）
f :: a -> b
g :: b -> c
-- 函数组合（证明）
g . f :: a -> c

-- 对应逻辑推理
a → b  (前提)
b → c  (前提)
─────────────
a → c  (结论)
```

### 哲学意义

> **程序即证明**：写程序就是在写证明

这使得我们可以通过类型系统来保证程序的正确性。

## 12.3.2.2 分布式一致性

### 定义

**控制结构（偏序） + 信息结构（容错逼近） ⇒ CAP 定理**:

CAP 定理本质上是"序"与"信息一致性"的权衡。

### CAP 定理的结构分析

| CAP 维度       | 结构类型 | 含义                               |
| -------------- | -------- | ---------------------------------- |
| **一致性**     | 信息结构 | 所有节点看到相同数据（信息一致性） |
| **可用性**     | 信息结构 | 系统持续可用（信息鲁棒性）         |
| **分区容忍性** | 控制结构 | 网络分区时的偏序（控制结构）       |

### 结构权衡

- **强一致性**（CP） ⇒ 强调信息结构，牺牲可用性
- **高可用性**（AP） ⇒ 强调信息结构，牺牲一致性
- **分区容忍性**（P） ⇒ 强调控制结构（偏序）

### 示例

```text
分布式数据库：
  - 一致性（信息结构）：所有节点看到相同数据
  - 可用性（信息结构）：系统持续可用
  - 分区容忍性（控制结构）：网络分区时的偏序

CAP 定理：
  三者不能同时满足，必须做出权衡
```

## 12.3.2.3 神经网络可解释性

### 定义

**信息结构（嵌入空间拓扑） + 计算结构（函数复合） ⇒ 决策边界分析**:

### 结构分析

| 组件         | 结构类型 | 含义                              |
| ------------ | -------- | --------------------------------- |
| **嵌入空间** | 信息结构 | 语义相近 ⇒ 距离近（拓扑保持映射） |
| **神经网络** | 计算结构 | 函数复合（前向传播）              |
| **决策边界** | 信息结构 | 嵌入空间的拓扑特征                |

### 示例

```text
词向量（信息结构）：
  vec("cat") 和 vec("dog") 的距离 < vec("cat") 和 vec("car") 的距离

神经网络（计算结构）：
  f(x) = W₃ · σ(W₂ · σ(W₁ · x))

决策边界（信息结构）：
  在嵌入空间中，决策边界是拓扑特征（分离超平面）
```

## 12.3.2.4 其他深层联系

### 1. 软件危机的根源

**三类结构失衡**：

- 重计算（功能），轻控制（并发 bug），弱信息（无抽象）

这导致：

- 功能丰富但并发 bug 多
- 缺乏抽象，代码重复
- 信息泄漏，安全问题

### 2. 量子计算

**新计算结构（非经典幺正变换） + 新信息结构（希尔伯特空间拓扑）**:

量子计算引入了新的计算结构和信息结构：

- **计算结构**：幺正变换（非经典计算）
- **信息结构**：希尔伯特空间拓扑（量子叠加）

### 3. AI 可解释性缺什么？

**缺乏"信息结构"的拓扑透明性**（黑盒无邻近性保证）

神经网络的黑盒性质源于信息结构的缺失：

- **计算结构**：函数复合（存在）
- **控制结构**：训练循环（存在）
- **信息结构**：拓扑透明性（缺失）
