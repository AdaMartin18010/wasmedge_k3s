## 1 文档定位

本文档提供案例研究中使用的**认知模型对比矩阵**，帮助理解不同认知模型的功能、适用场景、复杂度和学习路径，指导选择合适的认知模型进行案例分析。

**为什么需要认知模型对比矩阵？**

在分析案例研究时，我们面临以下挑战：

1. **模型选择困难**：多个认知模型，不知道选择哪个
2. **适用场景不清**：不清楚每个模型适用于什么场景
3. **学习路径不明**：不知道如何学习这些模型
4. **组合使用困惑**：不知道如何组合使用多个模型

认知模型对比矩阵通过**功能矩阵、场景矩阵、复杂度矩阵和学习路径矩阵**，帮助我们：

1. **快速对比**：快速了解不同模型的功能和特点
2. **选择模型**：根据问题选择合适的认知模型
3. **学习路径**：明确从新手到专家的学习路径
4. **组合使用**：了解如何组合使用多个模型

**核心价值**：

- **模型理解**：理解不同认知模型的功能和特点
- **场景匹配**：根据场景选择合适的模型
- **学习指导**：明确学习路径和难度
- **组合策略**：了解模型组合使用策略
