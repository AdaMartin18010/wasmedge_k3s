## 2 使用指南

### 2.1 快速认知

1. **从思维导图开始**：通过思维导图快速建立整体认知框架
2. **使用知识矩阵**：通过知识矩阵对比不同认知模型
3. **选择学习路径**：根据学习阶段选择合适的认知模型

### 2.2 深入学习

1. **理解核心理念**：深入理解理念导向、认知工具、演进主线、论证推理
2. **掌握理论视角**：掌握矩阵视角、代数结构视角、结构视角、调度视角
3. **应用认知模型**：在实际问题中应用认知模型
