### 2.2 快速认知

1. **从思维导图开始**：通过[认知框架思维导图](01-cognitive-framework-mindmap.md)快速建立整体认知框架
2. **使用知识矩阵**：通过[认知模型对比矩阵](02-cognitive-models-matrix.md)对比不同认知模型
3. **选择学习路径**：根据学习阶段选择合适的认知模型，参考[学习路径指南](06-learning-path-guide.md)

### 2.3 深入学习

1. **理解核心理念**：深入理解理念导向、认知工具、演进主线、论证推理，参考[核心理念应用综合指南](19-core-principles-application-guide.md)
2. **掌握理论视角**：掌握矩阵视角、代数结构视角、结构视角、调度视角，参考[认知模型集成关系图](05-cognitive-models-integration.md)
3. **应用认知模型**：在实际问题中应用认知模型，参考[认知模型完整工作流](10-cognitive-models-workflow.md)和[实践案例详细指南](16-practice-cases-detailed-guide.md)
