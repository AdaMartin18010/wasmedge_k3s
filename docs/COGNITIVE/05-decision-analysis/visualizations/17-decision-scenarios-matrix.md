**场景**：为多租户SaaS平台进行技术选型

**应用过程**：

1. **场景识别**：
   - 决策场景：技术选型场景
   - 场景类型：多租户SaaS

2. **场景匹配**：
   - 匹配场景：多租户SaaS场景
   - 需求：强隔离
   - 技术选择：Kata Containers
   - 认知模型：隔离模型+安全模型

3. **需求分析**：
   - 隔离需求：强隔离（多租户）
   - 安全需求：高安全性
   - 性能需求：中等性能

4. **方案参考**：
   - 参考场景选择Kata Containers
   - 理由：独立内核，强隔离

5. **决策制定**：
   - 选择Kata Containers
   - 使用LSM+SELinux进行安全隔离
   - 实施多租户隔离方案

**效果**：成功应用场景参考，选择合适的技术方案

    def identify_scenario(self, context, requirements):
