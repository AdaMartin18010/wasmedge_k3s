## 📋 清单概述

本文档提供**合规性检查清单**，涵盖 GDPR、HIPAA、等保等主要合规标准。

### 适用范围

- 数据处理和存储
- 用户隐私保护
- 安全控制措施
- 审计和日志记录

---

## 🇪🇺 GDPR 合规性检查清单

### 数据处理原则

- [ ] **合法性、公平性和透明性**
  - [ ] 数据处理有法律依据
  - [ ] 数据处理过程透明
  - [ ] 用户知情同意

- [ ] **目的限制**
  - [ ] 数据处理目的明确
  - [ ] 不超出原始目的范围

- [ ] **数据最小化**
  - [ ] 仅收集必要数据
  - [ ] 定期清理不必要数据

- [ ] **准确性**
  - [ ] 数据准确且及时更新
  - [ ] 提供数据更正机制

- [ ] **存储限制**
  - [ ] 数据存储时间有限
  - [ ] 定期删除过期数据

- [ ] **完整性和保密性**
  - [ ] 数据加密存储
  - [ ] 访问控制机制
  - [ ] 数据备份和恢复

### 数据主体权利

- [ ] **访问权**
  - [ ] 提供数据访问接口
  - [ ] 响应时间符合要求（30天）

- [ ] **更正权**
  - [ ] 提供数据更正机制
  - [ ] 及时处理更正请求

- [ ] **删除权（被遗忘权）**
  - [ ] 提供数据删除机制
  - [ ] 彻底删除数据

- [ ] **限制处理权**
  - [ ] 支持暂停数据处理
  - [ ] 保留数据但不处理

- [ ] **数据可携带权**
  - [ ] 提供数据导出功能
  - [ ] 支持标准格式导出

- [ ] **反对权**
  - [ ] 支持用户反对数据处理
  - [ ] 及时处理反对请求

### 技术措施

- [ ] **数据加密**
  - [ ] 传输加密（TLS 1.3+）
  - [ ] 存储加密（AES-256）
  - [ ] 密钥管理

- [ ] **访问控制**
  - [ ] 身份认证（MFA）
  - [ ] 权限管理（RBAC）
  - [ ] 最小权限原则

- [ ] **审计日志**
  - [ ] 记录所有数据访问
  - [ ] 日志不可篡改
  - [ ] 日志保留期符合要求

- [ ] **数据泄露通知**
  - [ ] 建立泄露检测机制
  - [ ] 72小时内通知监管机构
  - [ ] 及时通知受影响用户
