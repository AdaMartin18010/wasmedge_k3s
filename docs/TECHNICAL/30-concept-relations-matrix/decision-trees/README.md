## 使用说明

所有文档包含完整的决策树、决策矩阵和选型指南，可直接用于技术选型决策。

**使用建议**：

- **运行时选型**：根据场景需求选择 WasmEdge 或 runc
- **编排平台选型**：根据场景需求选择 K3s、K8s 或 Karmada
- **策略引擎选型**：根据 K8s 环境选择 Gatekeeper 或 OPA-Wasm
