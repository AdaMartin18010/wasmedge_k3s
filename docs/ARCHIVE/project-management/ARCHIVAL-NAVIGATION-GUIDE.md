## 📊 文档分类说明

### 任务执行相关（`task-execution/`）

包含任务执行进度、总结、完成报告等文档。这些文档记录了任务执行的完整过程。

**核心文档**：
- `PROJECT-TASKS-INCOMPLETE.md` - 项目未完成任务清单（当前活跃）
- `TASK-EXECUTION-PROGRESS.md` - 任务执行进度跟踪（当前活跃）
- `TASK-EXECUTION-GUIDE.md` - 任务执行指南（当前活跃）

### 工作完成报告（`work-completion/`）

包含各种工作完成报告和总结。这些文档记录了不同阶段的工作完成情况。

### 评估和改进报告（`evaluation-improvement/`）

包含项目评估、改进计划等文档。这些文档记录了项目的评估结果和改进方向。

**核心文档**：
- `IMPROVEMENT-ROADMAP.md` - 改进路线图（当前活跃）

### 认知增强相关（`cognitive-enhancement/`）

包含认知增强相关的工作报告。这些文档记录了认知增强工具补充的完成情况。
