## ✅ 结构检查清单

### 目录结构

- [ ] **目录标题**：使用 `## 📑 目录` 或 `## 目录`
- [ ] **目录格式**：使用 Markdown 链接格式
- [ ] **目录链接**：所有链接指向正确的章节锚点

### 章节编号

- [ ] **章节编号**：统一使用数字编号（如 `1.`, `1.1`, `1.1.1`）
- [ ] **编号一致性**：同级章节编号格式一致
- [ ] **编号连续性**：章节编号连续，无跳跃

### 代码块格式

- [ ] **语言标识**：代码块包含正确的语言标识（如 `bash`, `yaml`, `python`）
- [ ] **代码引用**：使用 `startLine:endLine:filepath` 格式引用现有代码
- [ ] **代码块缩进**：代码块不缩进（从列 0 开始）
